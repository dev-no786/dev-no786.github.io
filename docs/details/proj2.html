Multiplayer Socket based Game


Overview

Realtime shooter game using basic server-client architecture which I made from scratch using NodeJs, Express server and WS Websockets from client-server communication. About the game, it features an online deathmatch where can roam around and shoot enemies helped by the server which syncs all player's game states in realtime.
It also has a basic chat menu.

Server [ NodeJs | Websockets ]

Server is made by NodeJs & Express with Websockets for messaging over the network.

Client [ Unity | Websockets ]

The player client is made in Unity3d where the players connect to NodeJS server to join game.

Websocket Communication

Players commands are communicated over the server to other players in realtime using websocket.
