Online Poker Game


Overview

Through PlayerIO SDK I had managed to make this online game with features like -

  • Based Username/Password user login.
  • Lobby & Game Room Matchmaking.
  • Dedicated Game Database.
  • Server side scripting.
  • Player Leaderboard.

Basic Logging system

Simple Username Password account generation and login handled through PlayerIO account login system.

Lobby & Game Room Matchmaking

Players can join a games session with upto 3 players per room & have chance to leave or join new game.

Dedicated Game Database

Players client fetch their saved money from PlayerIO BigDB when playing to maintain secure access & persistent storage of player data

Dictating poker rules from server side

Server game logic decides the game result to ensure play by set rules & relay consistent data of their session to other clients in game , all the conditional logic of the card game is run on the server.
